## 🚀 Deployment Instructions

### Option 1: Web Hosting (Recommended)

1. Upload entire folder to your web host
2. Ensure PHP support (for contact form)
3. Update `forms/contact.php` email if needed
4. Test contact form functionality

### Option 2: GitHub Pages

1. Works for static pages
2. Contact form requires serverless solution (Firebase, Formspree, etc.)

### Option 3: Local Development

1. Use PHP built-in server: `php -S localhost:8000`
2. Or use XAMPP/WAMP for full PHP environment
3. See CONTACT_FORM_README.md for detailed setup

## 🎯 Customization Guide

### To customize this template:

1. **Replace Content**:
   - Update profile info in about.html
   - Add/modify portfolio projects in portfolio.html
   - Update skills and experience in resume.html
   - Modify services offered in services.html

2. **Update Contact**:
   - Edit email in `forms/contact.php` (line 30)
   - Update phone/location in contact.html

3. **Change Colors/Branding**:
   - Edit `assets/css/main.css`
   - Look for color variables and update as needed

4. **Replace Images**:
   - Add images to `assets/img/` folders
   - Update `src` attributes in HTML
